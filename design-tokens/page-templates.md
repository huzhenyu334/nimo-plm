# 页面模板规范 - nimo PLM/ERP

> 所有新页面必须基于以下三种模板之一。由 UX Agent 维护。

## 模板1：列表页（List Page）

适用于：项目列表、BOM列表、供应商列表等

```
┌─────────────────────────────────────────────┐
│ PageContainer (title + breadcrumb)          │
├─────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────┐ │
│ │ ProTable                                │ │
│ │ ┌─ Search Bar ────────────────────────┐ │ │
│ │ │ [搜索字段1] [字段2] [搜索] [重置]   │ │ │
│ │ └─────────────────────────────────────┘ │ │
│ │ ┌─ Toolbar ───────────────────────────┐ │ │
│ │ │                     [导出] [+ 新建] │ │ │
│ │ └─────────────────────────────────────┘ │ │
│ │ ┌─ Table ─────────────────────────────┐ │ │
│ │ │ 列1  │ 列2  │ 列3  │ 状态 │ 操作  │ │ │
│ │ │──────┼──────┼──────┼──────┼───────│ │ │
│ │ │ ...  │ ...  │ ...  │ Tag  │ 查看  │ │ │
│ │ └─────────────────────────────────────┘ │ │
│ │ ┌─ Pagination ────────────────────────┐ │ │
│ │ │           共x条  < 1 2 3 ... >      │ │ │
│ │ └─────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

## 模板2：详情页（Detail Page）

适用于：项目详情、BOM详情、供应商详情等

```
┌─────────────────────────────────────────────┐
│ PageContainer (title + breadcrumb + 返回)   │
├─────────────────────────────────────────────┤
│ ┌─ 基本信息 Card ──────────────────────────┐│
│ │ Descriptions                             ││
│ │ 名称: xxx    编码: xxx    状态: [Tag]    ││
│ │ 创建人: xxx  创建时间: xxx               ││
│ │                      [编辑] [删除]       ││
│ └──────────────────────────────────────────┘│
│ ┌─ Tabs ───────────────────────────────────┐│
│ │ [详细信息] [关联项] [操作记录] [附件]    ││
│ │ ┌────────────────────────────────────┐   ││
│ │ │ Tab内容区域                        │   ││
│ │ │ （表格/表单/Timeline等）           │   ││
│ │ └────────────────────────────────────┘   ││
│ └──────────────────────────────────────────┘│
└─────────────────────────────────────────────┘
```

## 模板3：表单页（Form Page）

适用于：新建/编辑项目、创建BOM等

```
┌─────────────────────────────────────────────┐
│ PageContainer (title + breadcrumb)          │
├─────────────────────────────────────────────┤
│ ┌─ 基本信息 Card ──────────────────────────┐│
│ │ Form.Item: 名称        [___________]     ││
│ │ Form.Item: 编码        [___________]     ││
│ │ Form.Item: 类型        [▼ 选择类型 ]     ││
│ │ Form.Item: 描述        [___________]     ││
│ │                        [___________]     ││
│ └──────────────────────────────────────────┘│
│ ┌─ 详细配置 Card ──────────────────────────┐│
│ │ Form.Item: ...                           ││
│ └──────────────────────────────────────────┘│
│ ┌─ Footer ─────────────────────────────────┐│
│ │              [取消]  [保存]              ││
│ └──────────────────────────────────────────┘│
└─────────────────────────────────────────────┘
```

## 通用规则

1. **间距**：Card之间 `margin-bottom: var(--spacing-md)` (16px)
2. **内边距**：Card内 `padding: var(--card-padding)` (24px)
3. **页面边距**：`padding: var(--page-padding)` (24px)
4. **最大宽度**：表单页内容区不超过 800px
5. **操作按钮**：主按钮靠右，"取消"在"保存"左边
