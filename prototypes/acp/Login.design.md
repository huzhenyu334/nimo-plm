# 登录页设计说明

## 布局

垂直水平居中，单卡片，宽度 380px。

```
┌─────────────────────┐
│                     │
│     🤖 (icon)       │
│  Agent Control Panel│
│  BitFantasy AI ...  │
│                     │
│  ┌─────────────────┐│
│  │ 🔒 输入管理密码  ││
│  └─────────────────┘│
│  ┌─────────────────┐│
│  │     登 录        ││
│  └─────────────────┘│
│                     │
│  v1 · 单管理员模式   │
└─────────────────────┘
```

## 关键决策

- **极简**：v1 仅密码认证，一个输入框 + 一个按钮
- **深色全屏背景** (#141414)：与整体深色主题一致
- **卡片背景** (#1f1f1f)：比页面底色略浅，形成层次
- **品牌感**：RobotOutlined 图标 + 蓝色强调色 + 产品名
- **Input.Password**：自带密码显隐切换

## 状态处理

| 状态 | 表现 |
|------|------|
| 空密码提交 | 表单校验红色提示 |
| 认证失败 | Alert 红色错误条 |
| 登录中 | 按钮 loading + 禁用输入 |
| 成功 | 跳转 `/`，token 存 localStorage |

## 安全
- token 存 localStorage（v1 简单方案）
- v2 改 httpOnly cookie + 飞书 SSO
